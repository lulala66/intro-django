# A polls app

## Environment

- Python 3.6.2
- Django 2.0.3

## Download this project

Use following command to download :ghost::

```bash
git clone https://github.com/lulala66/intro-django.git
```

## Install packages

Use pip install to install all required packages:

```bash
pip install requirements.txt
```
